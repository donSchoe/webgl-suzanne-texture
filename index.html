Full source is available on github.com/donschoe/webgl-suzanne-texture. The WebGL-code is inspired by the minimum-opengles-code.html by jay3sh and based on the code to draw simple lines by n.podbielski. The texture shading was made possible by the WebGL engine J3D by Bartek Drozdz. Used mat4.js from the glMatrix project by toji. The monkey Suzanne by SLiD3 used to be the free blender test model. Use github to raise issues or add comments to the code.
